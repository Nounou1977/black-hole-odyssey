## 📖 Description

**BlackHole Academy** est une plateforme éducative interactive dédiée à l'exploration des trous noirs et des mystères de l'univers. Notre mission est de rendre l'astronomie accessible à tous à travers du contenu gratuit, des quiz interactifs et des ressources pédagogiques de qualité.

🔗 **Site en ligne** : [BlackHole Academy](https://Nounou1977.github.io/blackhole-academy/)

---

## ✨ Fonctionnalités

- 📚 **Cours Complets** : Explications détaillées sur les trous noirs
- 🎯 **Quiz Interactifs** : 3 niveaux (Débutant, Intermédiaire, Avancé)
- 📰 **Blog Astronomique** : Articles récents sur les découvertes spatiales
- 📥 **Ebook Gratuit** : Guide PDF de 50 pages offert
- 📧 **Newsletter** : Abonnement pour recevoir les dernières actualités
- 💡 **Ressources Éducatives** : Glossaire, vidéos, infographies
- 📊 **Optimisé SEO** : Meta tags, sitemap, structure sémantique
- 💰 **Monétisation** : Espaces publicitaires Google AdSense

---

## 🛠️ Installation & Déploiement

### Déploiement sur GitHub Pages

1. Fork ce repository
2. Allez dans **Settings** → **Pages**
3. Source : **Deploy from a branch**
4. Branch : **main** → **/root**
5. Save et attendez 2-3 minutes
6. Votre site sera disponible à : `https://Nounou1977.github.io/blackhole-academy/`

### Déploiement sur Netlify

1. Créez un compte sur [Netlify](https://netlify.com)
2. Glissez-déposez le dossier sur [Netlify Drop](https://app.netlify.com/drop)
3. Votre site est instantanément en ligne !

### Déploiement sur Vercel

1. Installez Vercel CLI : `npm i -g vercel`
2. Dans le dossier du projet : `vercel`
3. Suivez les instructions

## 💰 Monétisation

### Google AdSense

5 espaces publicitaires sont prévus dans le code :
- Header banner (728x90)
- Sidebar (336x280)
- In-content (300x250)
- Footer banner (728x90)

**Pour activer** :
1. Inscrivez-vous sur [Google AdSense](https://adsense.google.com)
2. Obtenez vos codes publicitaires
3. Remplacez les espaces `<div class="ad-space">` par vos codes AdSense

### Collecte de Leads

Le système de quiz + landing page permet de :
- Capturer emails via formulaire
- Offrir un ebook gratuit
- Construire une liste d'abonnés
- Newsletter hebdomadaire

## 🎨 Personnalisation

### Couleurs

Le thème utilise une palette espace/galaxie :
- Violet : `#8a2be2`
- Bleu : `#4169e1`
- Cyan : `#00bfff`
- Fond sombre : `#0a0a1a` à `#1a0a2e`

### Polices

Police par défaut : `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif`

Pour changer, ajoutez dans `<head>` :
```html
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap" rel="stylesheet">
```

Et modifiez le CSS :
```css
body {
  font-family: 'Poppins', sans-serif;
}
```
